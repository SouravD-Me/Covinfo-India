About

Covinfo India is the final year project of four B.Tech Computer Science and Engineering students. This project is developed using Python 3.7.7 along with the packages for data analytics and visualization. We built and analyzed our dataset by collecting credible Covid-19 data from WHO Covid-19 dashboard for the latest case statistics in India, and Arogya Setu smartphone app for collecting the overall vaccination data in India. Finally, we embedded our data visulizations in a website using Matplotlib, HTML and CSS.
